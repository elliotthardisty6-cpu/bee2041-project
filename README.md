# BEE2041 Empirical Project
## Do Premier League Clubs Get Value for Money?
 
**Research question:** Does spending on wages lead to more league points,
and which clubs over- or under-perform for their wage bill?
 
## How to replicate this project
 
1. Clone this repository:
   git clone https://github.com/YOURUSERNAME/bee2041-project
 
2. Install required packages:
   pip install -r requirements.txt
 
3. Run the scripts in this order:
   python scripts/01_scrape.py
   python scripts/02_clean.py
   python scripts/03_analysis.py
   python scripts/04_plots.py
 
4. Open blog.ipynb in Jupyter Notebook to view the full blog post
 
## Data sources
- FBref.com: Premier League team statistics 2023/24 (scraped)
- Wage data: publicly reported estimates (see blog for full citation)
